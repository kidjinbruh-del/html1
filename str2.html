Типы VLAN

Существует несколько типов VLAN: Port-Based — на основе портов. Mac-Based — на основе MAC-адресов. Protocol-Based — на основе протоколов. Port-Based Каждый порт коммутатора явно назначается в определённую VLAN. Любое устройство, подключённое к такому порту, автоматически становится частью соответствующей виртуальной сети. Особенности: Применяется в пределах одного коммутатора, например, для организации нескольких рабочих групп небольшой сети. Конфигурация портов — статическая, может быть изменена только вручную. Каждый порт может входить только в одну VLAN. Mac-Based Членство в VLAN определяется по MAC-адресу устройства. Для работы такой схемы на коммутаторе должна быть настроена таблица соответствий «MAC-адрес → VLAN». Особенности: Обеспечивает мобильность устройств, но требует тщательного администрирования и редко используется в небольших сетях. Protocol-Based Принадлежность к VLAN определяется по типу сетевого протокола (например, IPv4, IPv6, IPX). Особенности: Коммутатор анализирует полученные пакеты без тега VLAN на порте и автоматически сопоставляет их с пользовательским шаблоном протокола. Если пакет соответствует шаблону, коммутатор добавляет соответствующий тег VLAN. Это позволяет, например, автоматически разделять трафик IP и IPX в сети.
